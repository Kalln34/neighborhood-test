# neighborhood-test

Project name: Neighborhood Navigator 

About: My project addresses the challenge of finding reliable, up-to-date information when moving to a new area. Neighborhood Navigator would serve as a centralized hub for this information, helping individuals become better informed about their new community. Designed to support moving preparedness, the platform would also continue to function as an ongoing reference for local resources and community information. This application is for families or individuals relocating due to work, students moving due to college, or for those who need to move due to life changes. Information that will be available includes healthcare services and hospitals, environmental and weather information, educational institutions, safety reports, transportation options, lifestyle amenities, and communtiy resources.


Link to live site: https://kalln34.github.io/neighborhood-test/

This application allows users to explore neighborhoods, view local insights, share tips or advice about locations, and save preferred locations. 

To begin, users can click “Explore a Location” on the home page or select “Explore Neighborhoods” from the navigation bar. From the Explore page, users can choose a state. For the purposes of this project, please select Connecticut.

On the State page, users can select a city to continue. Please choose Hartford to follow the full experience. A sidebar is also available to switch between states at any time.

On the City page, users can browse categories such as schools, healthcare, and attractions to view more detailed information. If a user is interested in the city, they can click “Save This City”, which will store the location in their Profile page.

The Local Insights page allows users to view and share community tips. Users can submit tips, filter by state and category, upvote or downvote posts, and delete tips.

On the Profile page, users can view their saved locations and manage them. While this page is designed to simulate a personalized account experience, full user authentication is not implemented in this version. Instead, saved data is stored locally to demonstrate functionality and user flow.

Overall, this project demonstrates how users can explore locations, gather insights, and track preferred areas in a simple and interactive way.
